# parallax
Parallax Scrolling Website

A modern and immersive parallax scrolling website built using HTML5, CSS3, JavaScript, and GSAP (GreenSock Animation Platform). This project focuses on creating smooth animations and engaging visual storytelling through dynamic scrolling effects.

Features
Smooth parallax scrolling effects
GSAP-powered animations
Scroll-triggered interactions
Modern and responsive design
Optimized animation performance
Engaging user experience

Technologies Used
HTML5
CSS3
JavaScript (ES6)
GSAP
ScrollTrigger Plugin
Learning Outcomes

Through this project, I gained practical experience with:
Advanced GSAP animations
ScrollTrigger implementation
Parallax design techniques
Performance optimization for animations
Interactive user interface development
Modern front-end development practices

Project Goals
The primary goal of this project was to explore creative web animation techniques and improve my understanding of interactive front-end experiences using GSAP.
